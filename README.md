## üöÄ Quick start

1. **Clone the repo.**

    ```shell
    git clone https://github.com/taham8875/youtube-home-page-clone.git
    ```

2. **Install dependencies.**

    ```shell
    cd youtube-home-page-clone
    npm install
    ```

3. **Start developing.**

    ```shell
    npm run dev
    ```

    Your site is now running at `http://localhost:5173`!

4. **Open the source code and start editing!**
